The next task is to fill the missing values.

missing<-is.na(Data)
n.missing<-sum(missing)

The total number of row that contain missing values are 2304. I chose to substitude the missing values with a random data point from the dataset.

set.seed(1)
random.imp <- function (a){
  missing <- is.na(a)
  n.missing <- sum(missing)
  a.obs <- a[!missing]
  imputed <- a
  imputed[missing] <- sample (a.obs, n.missing, replace=TRUE)
  return (imputed)
}
imputedSteps<-random.imp(Data$steps)
impData<-as.data.frame(cbind(imputedSteps,Data$date,Data$interval))
names<-colnames(Data)
colnames(impData)<-names
impData$date<-as.Date(impData$date,format="%Y-%m-%d",origin="1970-01-01")
Total<-by(impData$steps,impData$date,sum)
plot(Total~unique(impData$date),type="h", main = "Total number of steps per day", xlab="Date", ylab="Sum of steps", lwd = 7, col="gray")


Mean<-mean(Total)
Median<-median(Total)

The above plot is a copy of the firt one, but here missing values were imputed. As a result the dayly mean is 1.0769 × 104 and the dayly median is 1.0765 × 104. As a result the mean is almost the same(the diference is in the 5th decimal space), while the median has stayed the same.
